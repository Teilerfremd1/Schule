## Rechtspositivismus vs. Naturrecht

| Rechtspositivismus                                                       | Naturrecht                                                                                                                  |
| ------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------- |
| Naturrecht schützt nicht gegen Unrechtsstaaten                           | Wenn Gesetze Menschenrechte o.ä. einschränken, ist das ungerecht und **kein Recht**                                         |
| Jeder hat verschiedene Moralurteile und -vorstellungen                   | Man muss abwägen, wann die negativen Folgen des Gesetzes so schlimm sind, dass man dem Gesetz den Rechtscharakter abspricht |
| Antidemokraten könnten mit der "Moralkeule" gegen die Demokratie kämpfen | Es gibt o.g. Gesetze. Das Volk muss lernen, das zu sehen                                                                    |
| Keine Gewissheit, dass stets das Gute siegt                              | Es gibt Rechtsgrundsätze, die stärker sind als jede Satzung. Das sog. **Naturrecht**                                        |
| Begründung: Recht gilt, wenn formal korrekt (kein Problem)               |                                                                                                                             |
